# Agent Skill: SEO & Performance Optimization

## Purpose
Ensure websites are search-engine friendly, accessible, and fast-loading.

## Standards
1. **Semantic HTML**: Always use <header>, <nav>, <main>, <section>, <article>, and <footer>.
2. **Meta Tags**: Include title, description, and viewport meta tags.
3. **Accessibility (A11y)**: Use alt tags for all images and ensure sufficient color contrast.
4. **Heading Hierarchy**: Only one <h1> per page. Follow with <h2> and <h3> in logical order.
5. **Favicon**: Reference a generic high-quality favicon if none provided.

## Implementation Checklist
- [ ] Add <meta name="description" content="..."> with relevant project keywords.
- [ ] Use aria-label on interactive elements (buttons, links).
- [ ] Ensure all <a> tags have descriptive text (not "Click Here").
- [ ] Optimize loading by placing <script> at the end of <body>.
- [ ] Add a <title> that includes both the brand name and the page topic.
