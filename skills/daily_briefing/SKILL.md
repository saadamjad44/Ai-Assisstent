# Daily Briefing Skill

Generate automated daily briefings summarizing business activity.

## Overview

Every morning (scheduled via Task Scheduler), the system generates a CEO briefing containing:
- Tasks completed yesterday
- Pending tasks
- Items awaiting approval
- System health status
- Revenue tracking (if bank integration active)

## Briefing Location

Briefings are saved to: `/Briefings/briefing_YYYY-MM-DD.md`

## Manual Generation

```bash
python -c "from scripts.orchestrator import Orchestrator; o = Orchestrator('d:/practice/AI_Employee/AI-Employee-Vault'); o.generate_daily_briefing()"
```

## Scheduled Generation

Windows Task Scheduler runs this daily at 8:00 AM.

Setup script: `scripts/scheduled_tasks.bat`

## Briefing Format

```markdown
---
generated: 2026-02-07T08:00:00
period: 2026-02-07
---

# Daily CEO Briefing

## Summary
- **Tasks Completed Today:** 5
- **Pending Tasks:** 3
- **Awaiting Approval:** 1

## Completed Tasks
- [x] Client A invoice sent
- [x] Project Alpha milestone 2 delivered
- [x] LinkedIn post published

## Pending Items
- [ ] Review Client B proposal
- [ ] Approve payment request

## System Health
- Gmail Watcher: ✅ Online
- LinkedIn Watcher: ✅ Online
- Orchestrator: ✅ Running

---
*Generated by SAZA AI Employee*
```

## Customization

Edit `orchestrator.py` `generate_daily_briefing()` method to:
- Add revenue tracking (integrate with bank API)
- Include calendar events
- Track specific project milestones
- Add custom metrics

## Integration with Business Goals

The briefing can reference `/Business_Goals.md` to:
- Compare actual vs target metrics
- Flag overdue milestones
- Suggest proactive actions
