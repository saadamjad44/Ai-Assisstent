# Agent Skill: Interactive Elements (JS Components)

## Purpose
Add dynamic functionality to websites to enhance user engagement.

## Components
1. **Interactive Navbar**: Background color change on scroll, mobile hamburger menu.
2. **Smooth Scrolling**: Implement scroll-behavior: smooth and JS-based anchor scrolling.
3. **Form Validation**: Real-time feedback for contact forms.
4. **Modal Dialogs**: Premium popups for images or extra info.
5. **Dynamic Filters**: If applicable, implement JS-based filtering for project grids or galleries.

## Implementation Checklist
- [ ] Add 'active' class toggling for navigation links.
- [ ] Implement a 'Back to Top' button for long pages.
- [ ] Use Intersection Observer API for scroll-reveal animations.
- [ ] Ensure all forms have preventDefault() and a success message simulation.
- [ ] Use fetch() simulations for any dynamic data loading.
