# SAZA Store - Amazon Clone

A modern, responsive e-commerce website inspired by Amazon, built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works perfectly on mobile, tablet, and desktop devices
- **Modern UI**: Clean layout with Amazon-inspired design elements
- **Interactive Elements**: Hover effects, search functionality, cart system
- **Category Grid**: Easy navigation through product categories
- **Product Showcase**: Featured products with ratings and pricing

## Structure

```
Amazon_Pro_Clone/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # Stylesheet
├── js/
│   └── script.js       # JavaScript functionality
└── images/             # Placeholder images
```

## Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (ES6+)
- Font Awesome for icons

## Design Elements

- Primary Colors: #131921 (Header) and #febd69 (Buttons)
- Responsive Grid Layout
- Interactive Hover Effects
- Mobile-First Approach

## Functionality

- Dynamic search bar
- Shopping cart counter
- Product hover effects
- Category navigation
- Responsive breakpoints for all device sizes
